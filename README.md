# CrystalBuilder
Generate initial configurations of crystals

# Author
- Dr. Aristotelis P. Sgouros (arissgouros@gmail.com)

# Description
The code generates initial configurations of crystals

# Organization
The repository includes the following files and directories:
 - README           -> current file
 - LICENSE          -> the MIT LICENSE
 - example/         -> directory containing an indicative example
 - crystal_builder.py -> python file implementing the block averaging algorithm

# Note
You can run `crystal_builder.py` like a normal command by adding the repository's directory to your `PATH`. For example:

  export PATH="$PATH:/cyclonas_users/asgouros/programs/CrystalBuilder"

You can then invoke it simply as:

  crystal_builder.py
